## Comment Contribuer

### Signaler un Bug

1. **Vérifiez les issues existantes** - Assurez-vous que le bug n'a pas déjà été signalé
2. **Créez une nouvelle issue** - Utilisez le template de bug
3. **Incluez les détails** :
   - Description claire du problème
   - Étapes pour reproduire
   - Comportement attendu vs actuel
   - Captures d'écran si possible
   - Environnement (OS, navigateur, version)

### Proposer une Fonctionnalité

1. **Vérifiez les propositions existantes**
2. **Créez une nouvelle issue** - Utilisez le template de feature
3. **Décrivez la fonctionnalité** :
   - Problème résolu
   - Solution proposée
   - Alternatives considérées
   - Impact sur l'API existante

### Soumettre une Pull Request

1. **Forkez le repository**
2. **Créez une branche** :
   ```bash
   git checkout -b feature/nom-de-la-fonctionnalite
   # ou
   git checkout -b fix/description-du-bug
